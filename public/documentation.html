Documentation of "The Diversity Movie API"

The API is a Node.js and Express application and uses REST architecture. It returns data about movies with approach to diversity, equity and inclusion.

Business Logic URL HTTP Method Request body data format Response body data format
Get a list of all movies /movies GET None A JSON Object holding data about all movies
Get data about a single movie by title /movies/[title] GET None A JSON object holding data about a single movie, containing an title, description, genre, director, image URL
Get data about a genre (description) by name /genres/[name] GET None A text message
Get data about a director by name /directors/[name] GET None A JSON object holding data about a single director, containing bio, birth year and death year
Add a new user /users POST A JSON Object holding data about ths user to add, structured like: {username: "watcherOne", email: "erin.yilmaz@gmx.de", password: "jdhsk63%n", date of birth: "01.01.2001"} A text message indicating the username and the emailaddress of the user.
Update user info (username) /users/[username] PATCH None A text message indication the username that was updated
Add movie to favorites list /users/[username]/list POST A JSON object holding data about a single movie, containing an title, description, genre, director, image URL A text message including the title of the movie that has been added
Remove movie from favorites list /users/[username]/list/[title]/ DELETE None A text message including the title of the movie that has been deleted
Remove a user by username /users/[username] DELETE None A text message including the username and email that has been removed
