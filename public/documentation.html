An API in Node.js that returns facts about movies with an approach of diversity

Documentation

HTTP Method URL Business Logic Request body data format Response body data format

Example:

[ { "_id": "63596a61b2e4e7109ac44bb2", "title": "Black Panther", "Year": "2018", "Description": "After the death of his father...", "imageUrl": "https://url.com", "Featured": true, "directors": [ "635961ecb2e4e7109ac44b9a" ], "Genres": [ "635964e5b2e4e7109ac44ba4", "635964e5b2e4e7109ac44ba5", "635964e5b2e4e7109ac44ba6" ], "Actors": [ "63595eb0b2e4e7109ac44b6f", "63595f9db2e4e7109ac44b70", "63595f9db2e4e7109ac44b71" ] }, ]
GET /genres Get a list of all genres None An array holding JSON Objects

Example:

[ { "_id": "635964e5b2e4e7109ac44ba4", "Name": "Action", "Description": "Movies in the action genre are... " }, ]
GET /directors Get a list of all directors None An array holding JSON Objects

Example:

[ { "_id": "635961ecb2e4e7109ac44b9a", "Name": "Ryan Coogler", "birthYear": 1986, "bio": "Ryan Kyle Coogler is an African-American..." }, ]
GET /actors Get a list of all actors None An array holding JSON Objects

Example:

[ { "_id": "63595eb0b2e4e7109ac44b6f", "Name": "Chadwick Boseman" }, ]
GET /users Get a list of all users None An array holding JSON Objects

Example:

[ { "_id": "635cf4af71c8a5988bf0064b", "Username": "Tester7", "Password": "blablubi", "Email": "test5@web.de", "Birthday": "2007-03-20T23:00:00.000Z", "Favslist": [], "__v": 0 } ]
GET /movies/[title] Get data about a movie by title None A JSON object
GET /genres/[name] Get data about a genre by name None A JSON object
GET /directors/[name] Get data about a director by name None A JSON object
GET /users/[username] Get data about a user by username None A JSON object
POST /users Add a new user A JSON Object holding data about the user to add

Example:

{ "Username" : "watcherOne", "Password" : "zsvd3k/hs", "Email" : "user1@mail.de", "Birthday" : "2001-02-24" } 		A JSON Object holding data about the added user

Example:

{ "Username" : "watcherOne", "Password" : "zsvd3k/hs", "Email" : "user1@mail.de", "Birthday" : "2001-02-24" "Favslist": [], "_id": "635cf4af71c8a5988bf0064b", "__v": 0 }
PUT /users/[username] Update user info A JSON Object holding the updated line

Example:

{ "Username":"Tester7", } 		A JSON Object holding data about the updated user including the new data

Example:

{ "Username" : "Tester7", "Password" : "zsvd3k/hs", "Email" : "user1@mail.de", "Birthday" : "2001-02-24" "Favslist": [], "_id": "635cf4af71c8a5988bf0064b", "__v": 0 }
